# Component API

## Button
Props:
- `variant`: `solid | outline | ghost`
- `size`: `sm | md | lg`
- `href?`, `target?`, `onClick?`, `fullWidth?`

## Card
Props:
- `header?`, `footer?`, `elevation?`, `children`

## Text
Props:
- `as?`
- `variant`: `display | h1 | h2 | h3 | h4 | h5 | h6 | body | small | label`
- `tone`: `default | muted | accent | inverse`

## Stack
Props:
- `direction`: `row | column`
- `gap`: `xs | sm | md | lg | xl`
- `align?`, `justify?`, `wrap?`

## Container
Props:
- `size`: `sm | md | lg | xl`
- `padding`: `sm | md | lg`

## ThemeSwitcher
Props:
- `className?`
